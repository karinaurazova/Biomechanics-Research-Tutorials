# Loading history and time-scale separation

Growth is path dependent because the internal state integrates the complete stimulus history. Step, ramp, pulse and cyclic protocols can have the same peak stretch but different accumulated growth.

The model assumes a separation between fast elastic equilibration and slow growth. At each growth step the elastic state is evaluated as if mechanical equilibrium were established instantaneously. This quasi-static assumption is appropriate only when the mechanical relaxation time is much shorter than the growth time.

For cyclic loading, using an instantaneous stimulus, a cycle-averaged stimulus, a peak stimulus or a fatigue-like accumulation law gives different predictions. The tutorial uses the instantaneous history so learners can see the filtering role of slow growth, but it does not claim that this is the unique biological choice.
