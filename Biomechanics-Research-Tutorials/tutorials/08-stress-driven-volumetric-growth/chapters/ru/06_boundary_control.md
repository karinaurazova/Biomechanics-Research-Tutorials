# Тип управления является частью модели

При заданной деформации рост изменяет \(\mathbf F_e\) и способен релаксировать механический стимул. При идеальном заданном напряжении внешний регулятор удерживает сам стимул; рост изменяет полный размер вместо восстановления напряжения.

Для изотропного управления напряжением упругое растяжение \(\lambda_e\) определяется из нелинейного соотношения для напряжения Коши. Полное растяжение равно

\[
\lambda=J_g^{1/3}\lambda_e.
\]

Если заданное напряжение остаётся выше гомеостатической цели, чисто стресс-зависимый закон может непрерывно увеличивать \(J_g\), пока не будет достигнуто ограничение. Это не обязательно численная ошибка. Такой результат показывает, что режим управления нагрузкой и геометрия должны замыкать обратную связь.

Следовательно, выводы из испытаний с заданными перемещениями нельзя автоматически переносить на органы с заданной силой или давлением.
