# 08 Режимы отказа

Классический тензор возвращает одну доминирующую ось на окрестность. Если два семейства пересекаются с сопоставимой выраженностью, их произведения градиентов смешиваются до вычисления собственных значений. Результатом может стать промежуточная ось, пространственно меняющийся компромисс или низкая когерентность. Ни один из этих вариантов не восстанавливает оба семейства.

Другие режимы отказа: недостаточный контраст, элементы мельче масштаба производной, высокая кривизна, сильный низкочастотный фон, насыщение интенсивности, анизотропные пиксели и масштаб в пикселях без физической калибровки. Это нарушения предположений модели, а не только проблемы визуализации.
