# 07 Verification experiments

Verification begins with straight families spanning the full axial interval. It then moves to noisy fields, curved truth, sharp domain boundaries, illumination gradients, and a summary benchmark. The primary metrics are axial MAE, RMSE, median absolute error, P95, signed bias, and accepted-pixel coverage.

A useful benchmark should include both homogeneous and heterogeneous cases. Excellent performance on a uniform angle does not establish the ability to preserve boundaries or curvature. The committed CSV file can be regenerated and extended with new synthetic cases.
