# Experiment

Design a parameter study for the remodeling rate k.

1. Choose at least six positive values of k.
2. Define an adaptation-time metric, such as the first time at which the mismatch falls below 5 degrees.
3. Compute the analytical adaptation time for each k.
4. Estimate the same time from the numerical trajectories.
5. Plot both results against k and quantify their discrepancy.
6. Repeat one case with at least four time steps.
7. State whether the observed trend is consistent with the model equation.
8. Discuss numerical resolution for the largest value of k.

## Deliverable

A reproducible script, one figure, one verification table, and a 500–700 word interpretation structured as:

- question;
- method;
- result;
- verification;
- limitations.
