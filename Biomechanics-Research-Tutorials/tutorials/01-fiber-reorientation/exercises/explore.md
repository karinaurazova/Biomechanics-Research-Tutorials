# Explore

1. Run the baseline experiment and identify the initial angle, target angle, initial mismatch, and final mismatch.
2. Predict the characteristic time 1/k before inspecting the output.
3. Set the remodeling rate to zero and explain the result.
4. Replace the target angle of 30 degrees with 210 degrees. Explain why the trajectory should be unchanged.
5. Compare the Euler solution with the analytical solution for time steps 0.2, 0.05, and 0.01.
6. Run the orthogonal-case experiment. Explain why perturbations of opposite sign select different branches.

## Deliverable

A two-page note containing:

- one reproduced figure;
- a parameter table;
- one analytical calculation;
- a numerical verification statement;
- a 100-word explanation of the orthogonal-case ambiguity.
