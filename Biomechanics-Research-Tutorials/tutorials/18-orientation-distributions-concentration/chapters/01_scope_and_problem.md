# Scope and problem

Orientation distributions are an interface between images and mechanics. A segmentation mask may look acceptable, but if it rotates the dominant fibre axis, removes branches or changes local alignment, the mechanical prediction can change substantially.

This tutorial therefore compares representations of the same synthetic microstructure rather than unrelated datasets. The benchmark includes ground truth, SEM-like intensity, polarization-like orientation and a segmentation-derived mask. Because the true geometry is known, every modality can be compared against the same reference.
