# Схема датасета и иерархия верификации

Экспортированный NPZ содержит угловые состояния, шумные наблюдения, математические ожидания без шума, ориентацию, фазовую задержку, амплитуду, маску и JSON-происхождение. Unit tests проверяют алгебраические свойства, benchmark cases — сквозное восстановление, а для валидации потребуются будущие эксперименты.
