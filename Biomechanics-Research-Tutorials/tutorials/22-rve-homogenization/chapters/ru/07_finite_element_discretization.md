# Конечно-элементная дискретизация


RVE дискретизируется постояннодеформационными треугольными элементами. Каждая прямоугольная ячейка разбивается на два треугольника. Локальная матрица материала назначается из cell-wise поля микроструктуры.

Для каждого элемента код вычисляет strain-displacement матрицу, площадь и элементную жёсткость. Глобальная матрица собирается так же, как в tutorial по конечно-элементной модели роста, но теперь матрица материала меняется от элемента к элементу.

Микроскопическая неизвестная — это не полное поле перемещений, а периодическая флуктуация. Аффинная часть известна из заданной макроскопической деформации. Решение редуцированной системы даёт флуктуацию, минимизирующую микроскопическую упругую энергию при условии периодичности.

Так студент видит прямую связь между finite elements и homogenization: гомогенизация — это краевая задача со специальными boundary conditions и специальным постпроцессингом.
