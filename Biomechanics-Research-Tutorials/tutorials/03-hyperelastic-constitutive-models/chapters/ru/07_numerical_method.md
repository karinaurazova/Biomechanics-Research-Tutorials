# 07 — Численный метод и верификация

Для параметра пути \(q\) вычисляется

\[
R(q)=\frac{d}{dq}W(\mathbf F(q)).
\]

Используется центральная разность:

\[
R(q)\approx\frac{W(\mathbf F(q+h))-W(\mathbf F(q-h))}{2h}.
\]

## Аналитическая проверка

Для несжимаемой одноосной модели Нео-Гука

\[
P(\lambda)=\mu(\lambda-\lambda^{-2}).
\]

Численная производная сравнивается с этой формулой для разных удлинений и шагов.

## Объективность

При наложенном жёстком повороте

\[
\mathbf F^*=\mathbf Q\mathbf F,
\qquad
\mathbf C^*=\mathbf C,
\]

поэтому объективная энергия удовлетворяет

\[
W(\mathbf Q\mathbf F)=W(\mathbf F).
\]

Это свойство автоматически проверяется для всех шестнадцати моделей.

## Объёмные энергии

\[
U_q=\frac{K}{2}(J-1)^2,
\quad
U_{\log}=\frac{K}{2}(\ln J)^2,
\quad
U_{ST}=\frac{K}{4}(J^2-1-2\ln J).
\]

Большой объёмный модуль приближает несжимаемость, но может ухудшать обусловленность конечно-элементной задачи.
