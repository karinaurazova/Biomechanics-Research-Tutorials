# Random-feature PINN-like модель

Обычный PINN использует нейронную сеть и градиентную оптимизацию. В этом tutorial используется random-feature модель: поле перемещений записано как `u(x)=Phi(x)w`, где `Phi` фиксировано, а веса `w` находятся из линейной задачи.

Поскольку PDE residual линейна по `u`, каждое слагаемое loss становится блоком строк в одной least-squares системе. Это делает реализацию прозрачной и воспроизводимой.

Позже ту же структуру можно заменить на MLP, Fourier network или operator-learning модель.
