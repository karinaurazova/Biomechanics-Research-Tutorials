# Exercise 03 — Research challenge

Extend the model in one direction:

1. Replace the small-strain law with a finite-strain hyperelastic fibre model.
2. Estimate a spatially varying `fiber_scale(x)` instead of a global parameter.
3. Add a segmentation-bias experiment where the mask overestimates fibre fraction.
4. Connect the model to a small RVE solver.
5. Implement a simple Metropolis sampler and compare it with the Gaussian posterior.

Deliverable: a mini-report with assumptions, equations, code changes and limitations.
