# Constituents and shared observable motion

The observable deformation gradient is shared, but constituent elastic stretches differ because each cohort was deposited at a different time and with a different prestretch. Hence equal current kinematics do not imply equal constituent stress.

The generic example contains one primary load-bearing family, two off-axis collagen-like families, and an isotropic matrix. This minimal set can be reinterpreted: smooth muscle and collagen in a vessel, aligned cells and collagen in myocardium, collagen fascicles and matrix in tendon, or polymer fibers and hydrogel in an engineered construct.

A model should include only constituents that are needed to answer the research question. Adding poorly observable phases can increase parameter non-identifiability without improving predictive content.
