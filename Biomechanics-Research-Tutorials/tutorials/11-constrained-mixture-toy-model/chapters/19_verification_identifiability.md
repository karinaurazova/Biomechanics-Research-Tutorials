# Verification, observability, and identifiability

Verification begins with identities: positive mass, survival bounds, exact component-stress summation, homeostatic stationarity, and convergence with time step and history window. Only after these checks should biological interpretation be attempted.

Observability asks which hidden state affects each measurement. Identifiability asks whether different parameter sets can produce indistinguishable observations. The tutorial displays a mechanism–measurement map and a compensating-parameter valley.

Useful experimental design combines multiple loading modes, time points, structural imaging, composition measurements, and mechanical tests.
