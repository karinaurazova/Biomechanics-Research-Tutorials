# Scope and research question

Constrained-mixture theory is introduced as a bookkeeping framework for living load-bearing materials. The key question is not only how the tissue changes as a whole, but which constituent was produced, when it was deposited, how much survived, which natural configuration it carries, and how it contributes to the present mechanical state.

The tutorial is deliberately tissue-agnostic. The same state variables can represent collagen, elastin, smooth muscle, contractile cells, proteoglycans, reinforcing fibers, or engineered scaffold phases. Arteries, valves, myocardium, tendon, ligament, skin, airway, intestine, and tissue-engineered constructs appear as examples rather than as a privileged target.

All parameters and protocols are synthetic. The objective is to expose assumptions, verify identities, compare model classes, and identify which measurements would be required before any tissue-specific calibration.
