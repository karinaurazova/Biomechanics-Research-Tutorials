# Accumulation, degradation, and reversal

The tutorial compares increased fibrous production, accelerated degradation, and removal of the perturbation. Reversal is generally incomplete because old cohorts survive and newly deposited material carries a different natural state.

The same mathematical mechanisms can represent fibrosis-like accumulation, matrix loss, adaptive reinforcement, or scaffold degradation. Biological naming must follow evidence rather than curve shape alone.

A reversal experiment is especially informative because it exposes hidden memory that a single monotonic loading protocol cannot identify.
