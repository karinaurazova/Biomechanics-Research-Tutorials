# Results manifest

All outputs are synthetic teaching results generated by scripts in `experiments/`. English and Russian figures contain identical numerical data.

- `model_taxonomy`: model-class comparison.
- `mixture_architecture`: generic constituent architecture.
- `homeostatic_initialization`: baseline stationarity.
- `cohort_survival`: age-dependent survival.
- `deposition_stretch`: prestress at deposition.
- `stress_decomposition`: constituent load sharing.
- `pressure_overload` and `volume_overload`: transverse- and axial-dominated loading.
- `overload_comparison`, `reversal`, `fibrosis_hypertrophy`, `collagen_degradation`: alternative remodeling regimes.
- `history_dependence`, `cohort_vs_homogenized`, `history_truncation`: memory and reduction.
- `stability_map`, `half_life_sensitivity`: dynamic sensitivity.
- `polarimetry_bridge`: generic orientation-imaging priors; filename retained for backward compatibility.
- `observability_map`, `identifiability`, `benchmark_summary`: inverse-problem and verification diagnostics.
- `constrained_mixture.gif`: constituent-mass evolution.
