# The inverse opening-angle problem

In a forward problem, a stress-free sector, material law, and loads determine the closed ring. In an inverse problem, a measured opening angle and loaded geometry are used to infer an unknown reference state or material parameters.

The inverse problem is generally non-unique. Opening angle, shear modulus, axial prestretch, layer properties, and fiber reinforcement can compensate one another. One geometric endpoint therefore cannot identify the full residual-stress tensor.

The tutorial constructs loss surfaces and demonstrates parameter valleys. Additional pressure–diameter data, axial force, layer-resolved geometry, full-field displacement, and multiple cuts can improve identifiability.
