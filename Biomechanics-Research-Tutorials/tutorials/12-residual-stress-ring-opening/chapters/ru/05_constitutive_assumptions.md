# Определяющие предположения

Остаточное напряжение нельзя восстановить только по геометрии. Для секторной задачи требуется определяющий закон упругой аккомодации. В основной реализации используется несжимаемая неогуковская модель с дополнительным окружным армированием, работающим только на растяжение.

Альтернативы включают Mooney–Rivlin, экспоненциальные законы типа Fung, дисперсные модели Holzapfel–Gasser–Ogden, угловое интегрирование Ланира, вязкоупругость, активный тонус, повреждение и сжимаемые формулировки. Восстановленное поле напряжений может существенно зависеть от выбора модели.

Материальные параметры, единицы, конфигурация волокон, порог рекрутирования и обработка сжатия должны сообщаться вместе с углом раскрытия.
