# Workshop materials

The `workshop` directory converts repository tutorials into reusable master-class units.  Each package contains an instructor plan, a student workbook and a slide outline.  The materials reference the same code, data and figures as the corresponding tutorial, so teaching and independent reproduction stay aligned.

## Recommended format

- 90 minutes: use the motivation, live run and metric discussion sections.
- 120 minutes: add the controlled modification task.
- 180 minutes: include exercises, group interpretation and short reports.

## Teaching principle

Every workshop distinguishes verification from validation.  Synthetic ground truth is used to check algorithms and modelling assumptions; it is not presented as clinical or experimental validation.
