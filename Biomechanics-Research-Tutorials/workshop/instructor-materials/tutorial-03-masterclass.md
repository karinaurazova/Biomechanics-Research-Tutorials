# Master class 03 — Hyperelastic Constitutive Models

**Duration:** 150 minutes
**Format:** explanation, live coding, model comparison, group interpretation.

## Schedule

- 0–15 min — finite-strain motivation and the role of a strain-energy function;
- 15–35 min — kinematics, invariants, principal stretches, and objectivity;
- 35–60 min — isotropic model families;
- 60–80 min — live comparison across loading modes;
- 80–100 min — fibers, HGO, GOH, and tension-only recruitment;
- 100–120 min — myocardial material axes and shear planes;
- 120–135 min — numerical derivative and objectivity verification;
- 135–150 min — group model-selection challenge and conclusions.

## Required preparation

Participants should be able to run the repository environment and know basic NumPy and Matplotlib.

## Assessment prompt

Select a model for a synthetic tissue study and justify the choice using loading modes, symmetry, parameter identifiability, and validation strategy.
