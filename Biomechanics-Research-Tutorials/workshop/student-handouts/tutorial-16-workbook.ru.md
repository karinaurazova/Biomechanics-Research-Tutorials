# Рабочая тетрадь Tutorial 16

Для каждого эксперимента запишите параметры speckle, радиус подобласти, шаг, радиус поиска, СКО перемещения, долю валидных точек, окно сглаживания деформаций и наблюдаемый режим отказа. Объясните, почему высокий коэффициент корреляции может сочетаться со смещённой оценкой перемещения.
