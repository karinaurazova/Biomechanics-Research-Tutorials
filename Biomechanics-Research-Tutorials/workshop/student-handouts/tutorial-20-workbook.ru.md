# Рабочая тетрадь Tutorial 20 — Мультимодальный verification-ready synthetic benchmark

## Перед началом

Откройте папку tutorial:

```bash
cd tutorials/20-multimodal-verification-ready-benchmark
```

Прочитайте `README.ru.md` и определите входные данные, восстановленные величины и benchmark-выходы.

## Задание 1 — Запуск baseline

```bash
python reproduce.py
```

Зафиксируйте:

- основной файл датасета в `data/`;
- основную benchmark-таблицу в `data/`;
- два рисунка из `figures/`;
- одну анимацию из `animations/`, если она есть.

## Задание 2 — Интерпретация pipeline

Напишите три коротких утверждения:

1. Что известно точно, потому что данные синтетические?
2. Что оценивается методом?
3. Какой выход повлияет на downstream mechanics calculation?

## Задание 3 — Сравнение сценариев

Используйте benchmark-таблицу и рисунки, чтобы ответить:

- Какой сценарий даёт лучшую численную метрику?
- Какой сценарий даёт наиболее полезный для механики результат?
- Совпадают ли эти ответы?  Объясните почему.
