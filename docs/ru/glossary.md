# Глоссарий

[English](https://github.com/karinaurazova/Biomechanics-Research-Tutorials/blob/main/docs/glossary.md)

**Анизотропия**
Зависимость отклика материала от направления.

**Ориентация волокон**
Ось, связанная с семейством волокон. Для ненаправленного волокна углы, отличающиеся на 180°, соответствуют одной ориентации.

**Ремоделирование**
Изменение состава ткани, внутренней структуры или материальных свойств во времени.

**Верификация**
Проверка корректности реализации уравнений и численных алгоритмов.

**Валидация**
Оценка того, насколько модель адекватно представляет реальную систему для сформулированной цели.

**Синтетический benchmark**
Вычислительно сгенерированный тест с известными допущениями и ожидаемым поведением.

**Распределение ориентаций**
Плотность вероятности, описывающая относительную представленность осей волокон по углу.

**Средняя осевая ориентация**
Предпочтительная ненаправленная ось волокон; углы, отличающиеся на 180°, эквивалентны.

**Параметр концентрации**
Параметр, управляющий шириной выбранного распределения вероятности. Его смысл и диапазон зависят от определения распределения.

**Тензор ориентации**
Тензорный момент распределения ориентаций, используемый для компактного описания направленной структуры. Тензор второго порядка не сохраняет все особенности полного распределения.

**Угловое интегрирование**
Вычисление суммарного материального вклада путём интегрирования откликов отдельных ориентаций с весом плотности вероятности.

**Идентифицируемость**
Возможность однозначно определить параметры модели по доступным наблюдениям.

## Термины остаточных напряжений

**Остаточное напряжение** — напряжение, сохраняющееся после снятия внешних нагрузок.

**Остаточная деформация** — деформация, высвобождаемая разрезом или разделением материала.

**Угол раскрытия** — геометрическая мера раскрытия кольца; сам по себе не является напряжением.

**Формулировка начальных напряжений** — модель, использующая уже напряжённую отсчётную геометрию без явного построения ненапряжённого состояния.
