# Biomechanics Research Tutorials

**Biomechanics Research Tutorials** — воспроизводимый учебный репозиторий по вычислительной биомеханике мягких тканей.  Он содержит 25 самостоятельных tutorials, которые связывают биологическую мотивацию, математические модели, численные методы, Python-код, тесты, синтетические данные, визуализации и двуязычные учебные материалы.

## Для чего нужен этот репозиторий

Репозиторий задуман как учебная и исследовательская база.  Каждый tutorial можно использовать как короткое занятие, модуль мастер-класса, воспроизводимый мини-проект или основу для студенческой работы.  Модули сделаны универсальными: они не привязаны к одной ВКР, одной ткани или одному набору данных.

Все данные являются синтетическими, если в конкретном tutorial явно не написано иное.  Синтетика используется специально: в таких задачах заранее известны истинные маски, ориентации, материальные параметры, поля деформаций и benchmark-цели.

## С чего начать

Новый пользователь должен начать с [START_HERE.ru.md](START_HERE.ru.md).  Самый короткий путь:

```bash
python3 scripts/setup_venv.py
python scripts/verify_repository.py --strict
python scripts/run_reproduce_selected.py 01 13 17 22 25
```

Инструкции для Anaconda, VS Code, Visual Studio, Windows, Linux и macOS находятся в [docs/ru/development-environments.md](docs/ru/development-environments.md).

## Структура репозитория

```text
Biomechanics-Research-Tutorials/
├── tutorials/              # 25 самостоятельных учебных модулей
├── src/biomechanics_tutorials/
├── tests/                  # тесты уровня репозитория
├── docs/                   # документация на английском
├── docs/ru/                # документация на русском
├── assets/shared-figures/  # обзорные рисунки и GIF для релиза
└── scripts/                # setup, verification и reproduction helpers
```

Пакет `biomechanics_tutorials` является локальным пакетом этого репозитория, а не внешней зависимостью.

## Политика воспроизводимости

Каждый tutorial должен содержать:

- ясную математическую модель и допущения;
- воспроизводимую Python-реализацию;
- синтетические данные или явный код их генерации;
- рисунки и таблицы, пересоздаваемые через `reproduce.py`;
- тесты или численные sanity checks;
- учебные материалы на английском и русском.

Результаты tutorials являются образовательными вычислительными демонстрациями.  Их нельзя представлять как экспериментальную или клиническую валидацию, если отдельный модуль явно не содержит такую процедуру.
