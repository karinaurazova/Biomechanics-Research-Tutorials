# Experiment

Спроектируйте двухфакторное исследование по уровню шума и интеграционному масштабу. Заранее задайте критерий, объединяющий ошибку ориентации, покрытие и ширину размытия границы. Объясните, почему масштаб с минимальной MAE на однородном поле может быть неоптимальным для неоднородного изображения.
