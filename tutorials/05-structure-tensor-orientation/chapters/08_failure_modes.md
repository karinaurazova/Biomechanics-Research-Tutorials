# 08 Failure modes

The classical tensor returns one dominant axis per neighborhood. If two families cross with comparable strength, their gradient products are mixed before eigendecomposition. The result can be an intermediate axis, a spatially varying compromise, or low coherence. None of these is a reconstruction of both families.

Other failure modes include insufficient contrast, features below the derivative scale, very dense curvature, strong low-frequency illumination, clipping, anisotropic pixels, and a scale expressed in pixels without physical calibration. These are model-assumption failures, not merely visualization problems.
