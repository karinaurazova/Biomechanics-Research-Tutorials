# 02 Learning objectives

The learner should be able to derive the tensor, implement it with Gaussian derivatives, interpret eigenvalue-based coherence, and evaluate orientation with axial statistics. The learner should also be able to justify parameter choices rather than copying default values.

A successful analysis reports at least: pixel size or synthetic resolution, gradient scale, integration scale, angle convention, confidence mask, border treatment, error metric, and whether multiple local directions are expected.
