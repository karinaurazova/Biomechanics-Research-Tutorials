# Interpretation and limitations

The tutorial is deliberately reduced. It omits electrophysiology, spatial calcium propagation, detailed sarcomere states, viscoelasticity, compressibility constraints, three-dimensional boundary-value problems, and parameter fitting. The isotonic calculation is quasi-static and excludes velocity feedback in the equilibrium solve.

These omissions are features for teaching, but they must be stated. Verification of code is not validation of physiology.
