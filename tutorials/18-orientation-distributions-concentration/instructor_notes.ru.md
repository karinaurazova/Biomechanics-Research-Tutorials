# Методические заметки — Tutorial 18

Рекомендуемая длительность: 60–90 минут.

1. Начните с различия между направленными и осевыми данными.
2. Покажите обзор модальностей и спросите, какому представлению студенты доверяют больше.
3. Рассчитайте ODF и параметры концентрации.
4. Обсудите, почему визуально небольшая ошибка сегментации может заметно изменить downstream-жёсткость.
5. Завершите переходом к Tutorial 19, где эти величины станут входами для идентификации параметров определяющих соотношений.
