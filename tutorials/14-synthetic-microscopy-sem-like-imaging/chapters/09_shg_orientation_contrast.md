# SHG-like orientation-sensitive contrast

SHG microscopy can provide high-contrast label-free views of ordered structural proteins and has been used extensively for collagen architecture \cite{campagnola2002,pijanka2019}. The tutorial uses a deliberately simplified axial factor $I\propto A^2|\cos(	heta-\phi)|^p$. It is useful for testing orientation pipelines but is not a full nonlinear-susceptibility or phase-matching model.
