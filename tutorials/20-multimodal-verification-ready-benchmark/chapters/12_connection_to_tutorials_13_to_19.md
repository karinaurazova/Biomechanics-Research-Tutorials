# Connection to Tutorials 13–19

Tutorial 20 synthesizes the image-informed block. Tutorial 13 generated synthetic fibrous tissue; 14 introduced microscopy-like imaging; 15 handled polarization-like orientation; 16 introduced synthetic DIC; 17 compared segmentation; 18 quantified orientation distributions; 19 identified image-informed material parameters.

The pedagogical arc is intentional: students first understand each stage separately, then see how errors propagate through the coupled workflow.

The next tutorials can move to advanced methods: finite-element growth, RVE homogenization, surrogate modeling, physics-informed learning and uncertainty analysis.

The chapter should be read together with the code. Every quantity mentioned here is created explicitly in `multimodal_benchmark.py` and regenerated by `reproduce.py`. The purpose is not to hide complexity behind a black box, but to show which assumptions enter the pipeline.

For interpretation, always ask four questions: what is the hidden truth, what is observed, what is reconstructed, and which metric compares them? This four-part logic prevents a visually appealing result from being mistaken for a verified mechanical conclusion.

## Implementation details

The tutorial imports ideas from the previous image-informed modules but keeps the code self-contained. This makes it possible to run Tutorial 20 without manually chaining scripts from Tutorials 13–19, while still preserving their conceptual sequence. The result is a compact final benchmark for the whole block.

## What to check while reading

Read this module after reviewing at least the README files for Tutorials 13–19. Then map each output of Tutorial 20 to an earlier module: synthetic structure, microscopy-like image, polarization angle, DIC strain, segmentation metric, ODF/concentration idea and image-informed parameter estimate. This mapping shows why the roadmap is cumulative.

## Typical misunderstanding

The final module is not a replacement for the earlier tutorials. It is a synthesis. The earlier modules teach the individual tools; Tutorial 20 teaches how those tools interact and how their errors propagate toward mechanics.
