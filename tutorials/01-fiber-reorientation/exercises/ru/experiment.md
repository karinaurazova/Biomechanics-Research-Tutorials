# Experiment

Разработайте параметрическое исследование скорости $k$.

1. Выберите не менее шести положительных значений $k$.
2. Определите метрику времени адаптации, например первое время, когда рассогласование меньше 5°.
3. Вычислите аналитическое время для каждого $k$.
4. Оцените его по численным траекториям.
5. Постройте оба результата и количественно сравните их.
6. Повторите один случай как минимум для четырёх шагов времени.
7. Объясните соответствие тренда уравнению модели.
8. Обсудите разрешение для наибольшего $k$.

## Результат

Воспроизводимый скрипт, один рисунок, таблица верификации и интерпретация объёмом 500–700 слов со структурой: вопрос, метод, результат, верификация, ограничения.
