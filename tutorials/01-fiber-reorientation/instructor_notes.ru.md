# Методические заметки преподавателя

## Основные ошибочные представления

1. **Угол волокна является углом направленного вектора.** Покажите эквивалентность 10° и 190°.
2. **Целевое направление получено из биологических измерений.** Здесь оно задано; в следующих модулях его можно вычислять из напряжений или деформаций.
3. **Гладкий график подтверждает модель.** Он показывает только поведение выбранного уравнения.
4. **Случай 90° является обычным.** Без дополнительного правила он не имеет единственного решения.
5. **Устойчивый шаг автоматически является точным.** Требуйте сравнение при уменьшении шага.

## Рекомендуемая последовательность live coding

1. Построить наивную разность углов.
2. Показать её ошибку около границы 180°.
3. Ввести удвоенный угол.
4. Вывести экспоненциальное решение для постоянной цели.
5. Реализовать явный метод Эйлера.
6. Проверить нулевую скорость и осевую эквивалентность.
7. Сравнить численную и аналитическую траектории.
8. Показать неоднозначность ортогонального случая.
9. Выполнить параметрический sweep.
10. Завершить обсуждением границ доказательности и валидации.

## Контрольные точки

- 25-я минута: вычисление осевой разности вручную;
- 55-я минута: прогноз знака шага Эйлера;
- 80-я минута: оценка времени адаптации до запуска кода;
- 105-я минута: одна формулировка о верификации и одна — об ограничении валидации.
