# Тестирование и воспроизводимость

Тесты проверяют генерацию данных, положительность жёсткости, уменьшение невязки, sweep по весу physics loss и обратную калибровку. Они не доказывают биологическую истинность, а подтверждают внутреннюю согласованность computational experiment.

`reproduce.py` заново генерирует основные таблицы и figures с фиксированным seed. Поэтому tutorial подходит для занятий и GitHub regression testing.

Студентам стоит менять noise level, observation density и image-prior bias, чтобы увидеть границы устойчивости метода.
