# 05 Упругий отклик и меры напряжений

Упругое определяющее соотношение вычисляется по \(\mathbf F_e\), а не по полной \(\mathbf F\). Используется сжимаемая неогуковская энергия на единицу объёма выросшей ненапряжённой конфигурации,

\[
W(\mathbf F_e)=
\frac{\mu}{2}
\left(I_{1e}-3-2\ln J_e\right)
+
\frac{\kappa}{2}(\ln J_e)^2.
\]

Для энергии на единицу исходного отсчётного объёма принята конвенция

\[
\Psi(\mathbf F,\mathbf F_g)=J_g W(\mathbf F_e).
\]

Отсюда

\[
\mathbf P=J_g\mathbf P_e\mathbf F_g^{-T},
\]

где \(\mathbf P_e=\partial W/\partial\mathbf F_e\). В литературе встречаются другие конвенции отсчётного объёма, поэтому выбор должен быть явно указан: от него зависит номинальное напряжение.

Напряжение Коши определяется упругой частью,

\[
\boldsymbol\sigma=rac{1}{J_e}\mathbf P_e\mathbf F_e^T.
\]

Напряжение Манделя

\[
\mathbf M=\mathbf F_e^T\mathbf P_e
\]

в ряде термодинамических формулировок сопряжено скорости роста и потому естественно выступает механическим стимулом.

Обязательны две проверки. Наложенный жёсткий поворот текущей конфигурации не должен изменять энергию. Кроме того, конечно-разностная производная \(\Psi\) по \(\mathbf F\) должна совпадать с реализованным \(\mathbf P\). Обе проверки автоматизированы.
