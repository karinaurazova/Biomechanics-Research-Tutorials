# 08 Homogeneous stress-driven adaptation

A homogeneous test is the correct place to debug a growth law before adding spatial equilibrium. The tutorial prescribes a fixed total stretch and lets \(\mathbf F_g\) evolve. As growth accumulates in the overloaded direction, \(\mathbf F_e=\mathbf F\mathbf F_g^{-1}\) approaches a lower-stress state.

The experiment records:

- principal growth stretches;
- components of Mandel and Cauchy stress;
- reference energy;
- \(J\), \(J_e\), and \(J_g\);
- the growth-driving contraction \(\mathbf M:\mathbf L_g\).

Changing the rate parameter changes the adaptation time but should not change the target equilibrium in a simple monotone law. Changing the target Mandel stress changes the residual elastic state. If a response limit becomes active, the transient is no longer a simple exponential.

The isotropic projection uses only the mean stress error. A diagonal projection allows independent material-axis growth. A full symmetric projection can generate shear components of \(\mathbf F_g\). These modes are not interchangeable and should be selected based on tissue architecture and the intended biological mechanism.

A decrease of elastic energy under one prescribed loading path is useful evidence but not a complete stability proof. The same law must be tested under unloading, multiaxial loading, rotated axes, noise, and parameter variation.
