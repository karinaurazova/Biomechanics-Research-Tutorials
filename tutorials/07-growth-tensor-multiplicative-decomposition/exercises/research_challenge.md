# Research Challenge

Extend the incompatible-growth example to a genuine boundary-value problem.

Possible directions:

1. solve a two-dimensional elastic equilibrium problem with a prescribed spatial `Fg` field;
2. compare a row-wise-curl diagnostic with a metric-curvature incompatibility measure;
3. implement a weak form for `Psi(F, Fg) = Jg W(F Fg^{-1})`;
4. verify the residual and consistent tangent by finite differences;
5. study mesh convergence and growth-induced buckling;
6. compare the morphoelastic result with a reduced beam or shell theory.

The final report must separate code verification, parameter sensitivity, and biological interpretation.
