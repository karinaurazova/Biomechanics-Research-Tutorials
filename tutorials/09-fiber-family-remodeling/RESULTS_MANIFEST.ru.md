# Манифест результатов Tutorial 09

Все рисунки и анимации создаются скриптами из `experiments/`. Английские и русские версии используют одни численные данные и различаются только локализованными подписями. Полное воспроизведение:

```bash
python tutorials/09-fiber-family-remodeling/reproduce.py
```

Основные группы результатов: таксономия подходов, прямая ротация, вырождение главного направления, эволюция ODF, сравнение Ланира и структурного тензора, HGO-подобные семейства, рекрутирование, когортный turnover, растяжение отложения, конкуренция семейств, идентифицируемость и synthetic benchmark.
