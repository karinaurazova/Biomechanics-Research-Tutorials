# Generic multi-tissue examples

The same four-phase teaching model is interpreted for several tissues. In an artery, the primary family can represent smooth muscle and the off-axis families collagen. In myocardium it can represent myocytes and collagen. In tendon or ligament, the primary family represents aligned fascicles while the matrix controls transverse response. In an engineered construct, the phases can represent reinforcing fibers and hydrogel.

These interpretations require different parameter ranges, stimuli, boundary conditions, and validation data. Reusing the code does not imply that the biology is identical.

A comparison table in the notebook helps learners decide which constituents, observables, and time scales are relevant for each application.
