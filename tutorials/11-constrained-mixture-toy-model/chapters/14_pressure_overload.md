# Transverse-dominated loading

A transverse-dominated protocol increases demand in off-axis families. It can represent pressure loading of a tube, transverse confinement, biaxial stretching, or another perturbation that preferentially loads circumferential or cross-fiber constituents.

The example demonstrates collagen-like production, load redistribution, and delayed remodeling. It should not be interpreted as a unique organ-level phenotype.

Changing boundary control from prescribed deformation to prescribed traction can reverse feedback behavior, so the control mode is documented explicitly.
