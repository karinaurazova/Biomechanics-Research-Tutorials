# Гомогенизированное constrained-mixture приближение

Гомогенизированная модель заменяет явный список когорт небольшим набором усреднённых внутренних переменных, например общей массой и средним естественным растяжением. Это позволяет уменьшить число состояний с порядка Ntime × Ncomponents до порядка Ncomponents.

Приближение работает, когда свойства когорт изменяются слабо, а выбранные моменты сохраняют существенную часть механической истории. Оно может нарушаться при резких изменениях нагрузки, широком возрастном распределении, нелинейном рекрутировании, повреждении или сильно меняющемся deposition stretch.

В tutorial траектории сравниваются количественно; редукция не объявляется автоматически эквивалентной полной истории.
