# Mechanobiologically equilibrated initialization

A simulation should not drift before the perturbation unless such drift is intentional. Homeostatic initialization therefore calibrates constituent targets and basal production/removal at the chosen baseline deformation.

The tutorial checks mass stationarity, stress decomposition, and target consistency. It also demonstrates that many different hidden states can reproduce the same baseline total stress.

Initialization is part of the inverse problem. It should be reported with the same care as material parameters and boundary conditions.
