# Упражнение 02 — Изменить обратную задачу

Измените один элемент benchmark:

- уберите shear load case;
- увеличьте шум силы;
- сделайте поле ориентаций почти однородным;
- уменьшите контраст fibre density;
- уберите virtual-field equations.

Повторите расчёт и сравните condition number, ширину posterior interval и ошибку параметров.

Результат: одна таблица и один интерпретационный абзац.
