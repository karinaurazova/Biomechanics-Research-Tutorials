# 03 — Image-informed определяющее соотношение

Используется малодеформационная анизотропная модель с матричным и волоконным вкладом. Напряжение записывается в Voigt notation:

```text
sigma = [sigma_xx, sigma_yy, sigma_xy]
```

Матричный отклик — изотропная plane-stress модель на единицу модуля. Волоконный отклик действует вдоль локальной оси `n = [cos(theta), sin(theta)]`.

Локальная деформация вдоль волокна:

```text
e_f = n_x^2 epsilon_xx + n_y^2 epsilon_yy + n_x n_y gamma_xy.
```

Волоконный вклад в напряжение пропорционален

```text
e_f [n_x^2, n_y^2, n_x n_y].
```

Вектор image-informed параметров:

```text
p = [matrix_base, matrix_density_gain, fiber_scale, connectivity_scale].
```

Модель можно записать как

```text
sigma(x) = B(x, epsilon, theta, kappa, rho_f, connectivity) p.
```

Это центральный методический выбор. Модель линейна по неизвестным параметрам, но нелинейно зависит от структурных полей из изображения. Поэтому её можно строго проверить, сохранив при этом связь структуры и механики.
