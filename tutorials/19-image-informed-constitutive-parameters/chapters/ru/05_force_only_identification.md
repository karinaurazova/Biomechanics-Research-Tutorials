# 05 — Идентификация только по силе

Самая простая калибровка использует только глобальные компоненты силы. Для каждого load case модель предсказывает

```text
y_i = average(sigma_i) = average(B_i) p.
```

После объединения всех load cases получается линейная система:

```text
A p ≈ y.
```

В туториале она решается методом non-negative least squares. Неотрицательность не является универсальным законом, но в учебной задаче она предотвращает физически бессмысленные отрицательные жёсткости.

Force-only calibration полезна, но ограничена. Она может восстановить параметры при разнообразных режимах нагружения, однако становится плохо обусловленной, если наблюдения слабо чувствительны к одному параметру или если два параметра компенсируют друг друга.
