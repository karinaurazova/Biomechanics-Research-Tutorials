# 06 — Virtual-field-style уравнения

Full-field strain data не стоит сразу сводить к одному глобальному числу. Virtual fields method использует взвешенные формы равновесия, чтобы извлечь дополнительную информацию из пространственно неоднородных strain fields.

В туториале используются гладкие virtual weights для левой, правой, верхней, нижней и центральной областей. Для каждого веса и load case собирается weighted stress observation:

```text
weighted observation = average(w(x) sigma(x)).
```

В реальном VFM правая часть получается из measured boundary work и выбранных virtual displacement fields. В синтетическом туториале она создаётся по известной модели и затем зашумляется. Это позволяет проверить inverse pipeline, сохранив компактный код.

Главная идея — identifiability: пространственное взвешивание помогает разделить параметры, которые выглядят похожими при глобальном усреднении.
