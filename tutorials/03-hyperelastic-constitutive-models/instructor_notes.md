# Instructor notes — Tutorial 03

Recommended duration: 150–180 minutes.

The central teaching message is not that one model is universally best. The message is that model structure, deformation protocol, parameter identifiability, and verification must be considered together.

## Suggested sequence

1. Introduce finite-strain kinematics and energy-based modelling.
2. Compare the ten isotropic laws in a narrow uniaxial interval.
3. Reveal divergence in biaxial loading and shear.
4. Introduce fiber invariants and tension-only recruitment.
5. Discuss myocardial material axes and shear-plane data.
6. Run the derivative and objectivity checks.
7. End with a model-selection discussion based on the intended application.

## Common misconceptions

- A curve fit is not a constitutive validation.
- A parameter with the same symbol in two models need not have the same meaning.
- Incompressibility is a constraint or approximation, not a synonym for softness.
- Fiber dispersion parameter κ in GOH is not the same as the axial von Mises concentration β used in Tutorial 02.
- Numerical differentiation verifies implementation consistency but does not prove biological validity.
