# 05 — Волокнистые модели

Для направления \(\mathbf a_0\) вводится структурный инвариант

\[
\bar I_4=\mathbf a_0\cdot\bar{\mathbf C}\mathbf a_0.
\]

## Экспоненциальное семейство только при растяжении

\[
W_f=\frac{k_1}{2k_2}
\left[\exp\left(k_2\langle \bar I_4-1\rangle_+^2\right)-1\right].
\]

Скобка Маколея исключает идеализированный вклад коллагена при сжатии.

## HGO

Два симметричных семейства расположены под углами \(+\alpha\) и \(-\alpha\). Изменение угла перестраивает включение волокон при фиксированном направлении нагрузки.

## GOH с дисперсией

\[
E_\kappa=\kappa(\bar I_1-3)+(1-3\kappa)(\bar I_4-1),
\qquad 0\le\kappa\le\frac13.
\]

При \(\kappa=0\) семейство идеально выровнено, а при \(\kappa=1/3\) его второй момент становится изотропным.

Параметр \(\kappa\) GOH не совпадает с концентрацией \(\beta\) осевого распределения фон Мизеса из Tutorial 02.
