# Instructor notes — Tutorial 25

This module is best taught after students have seen the earlier forward-model tutorials. The key pedagogical message is that uncertainty quantification is not an optional add-on at the end of a project. It changes how conclusions are stated.

Recommended flow:

1. Start from the parameter table and ask which ranges are most debatable.
2. Run Monte Carlo and discuss output intervals before looking at sensitivity indices.
3. Compare tornado, Morris and Sobol rankings. Emphasize that disagreement is informative, not a failure.
4. Use the posterior update to discuss identifiability and the danger of overconfident calibration.
5. Finish with the reporting checklist: ranges, sample count, output intervals, sensitivity metrics and limitations.
