# Треугольный элемент с постоянной деформацией


В реализации используется constant-strain triangular element. У каждого треугольника три узла, а в каждом узле две степени свободы перемещения. Поле перемещений внутри элемента интерполируется линейно. Поэтому strain внутри треугольника постоянен.

Это не самый точный конечный элемент. Он может быть слишком жёстким в задачах изгиба и грубо описывает градиенты. Его преимущество — прозрачность. Матрица strain-displacement B выписывается явно через координаты вершин. Элементная матрица жёсткости равна area times B transpose C B. Студент может распечатать эту матрицу и понять происхождение каждого члена.

Вектор деформаций хранится как [epsilon_xx, epsilon_yy, gamma_xy]. Компонента shear — это engineering shear gamma_xy = du_x/dy + du_y/dx. Матрица материала записана согласованно с этой конвенцией. Смешивание tensor shear и engineering shear — частая причина ошибки с фактором два.

Эквивалентная нагрузка роста равна area times B transpose C epsilon_g. Знак следует из sigma = C(epsilon - epsilon_g). Если система записана как K u = f_g, рост оказывается в правой части. Алгебраически это похоже на thermal strain load в линейной термоупругости.

Сетка структурирована, но assembly написан как общий цикл по элементам. Это облегчает будущий переход к неструктурированным сеткам. Element routine отделён от global assembly, чтобы его можно было тестировать отдельно.
