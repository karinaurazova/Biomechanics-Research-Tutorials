# Synthetic dataset schema

The exported NPZ file stores reference and deformed images, exact displacement components, and JSON metadata. Metadata explicitly states that the data are synthetic and that no experimental validation is claimed. Units, random seed, pixel size, loading mode, and forward-model settings can be added without changing the array schema.

This structure supports future tutorials on inverse material identification, multimodal benchmarking, segmentation-aware masks, and physics-informed learning.
