# Рабочая тетрадь Tutorial 08

Для каждого расчёта фиксируйте: меру напряжения, соглашение о знаках, цель, масштаб, тип управления, шаг времени, настройки роста/резорбции, активированные границы, конечное среднее напряжение, конечное напряжение Мизеса и конечное Jg.

Не называйте траекторию гомеостатической, пока явно не указана регулируемая величина.
